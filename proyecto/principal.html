Weather analysis



"What's the weather like as we approach the equator?" Now, we know what you may be thinking: "Duh. It gets hotter..." But, if pressed, how would you prove it? On this analisys we can examine how the proximity to the equator affects:

  • Temperature_max
  • Humidity
  • Cloudiness
  • Wind Speed

We obtained the information from the API: Open weather from which with the help of a process in python we downloaded the information from approximately 500 places, with the generation of data in a range of -90 to 90, in which we take latitude and longitude, and the name of each of the places, to be able to access the data mentioned above.
